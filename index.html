My projects

Quizbot - SMS Quiz Generator

This is an open source project that I felt passionate about. I have always found SMS texting and chat bots interesting and one day while sitting in a waiting room I thought, here we have a great opportunity to engage people - a captive audience. So I decided to make a quiz framework so that other people could replicate this project and just change-up the seed file to tailor it to their quiz theme.

Movietime - API exploration

This application makes searching for TV programs, movies and actors, fun! This is a full-stack application that leverages a community api: MovieDB. This quick project allowed me to explore API connectivity and integration in the rails environment.
